# UK Job Pathway - School Project Demo

> **DISCLAIMER**: This is a mock website created for a school project. It is NOT a real job platform and should NOT be used for actual job applications, immigration advice, or any other real-world purposes.

## About This Project

This website was created as part of a school project to demonstrate web development skills. It showcases a concept for helping immigrants to the UK find jobs and understand the employment landscape.

**All job listings, companies, and features are FICTIONAL.**

## Features (Demo Only)

- Job listings with filtering
- Qualification recognition chatbot (4 languages)
- Application tracking dashboard
- Mock video interview system
- Career resources section
- AI career assistant

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Lucide React for icons

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
